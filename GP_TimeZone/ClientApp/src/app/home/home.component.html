Wellcome to GP Time Zone

This a single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

We are using the API : https://www.timeapi.io/swagger/index.html

With this API done with Angular and WebAPICore.NET you can get the current time of a time zone all over the world.
